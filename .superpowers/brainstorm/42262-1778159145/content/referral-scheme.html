🔄 Referral Flow (Client & Mitra)

User Journey × Referral Milestones
📱 Referrer share kode → 👤 Referee daftar (pakai kode) → ✅ Milestone 1: Registered
↓ Reward: Voucher diskon 10% untuk referee
✅ Milestone 2: Verified (Client: email verified | Mitra: dokumen approved)
↓ Reward: Rp 10.000 cashback ke wallet referrer
✅ Milestone 3: First Action (Client: deposit pertama | Mitra: transaksi pertama)
↓ Reward: Rp 25.000 cashback ke wallet referrer + Rp 15.000 ke referee

💰 Reward Types (Configurable per Milestone)

Wallet Cashback

• Langsung masuk saldo wallet

• WalletTransaction type: referral_reward

• Reference: referral record ID

• Nominal configurable per campaign

Voucher Auto-Assign

• Auto-assign voucher ke user

• Leverage VoucherUser existing

• Voucher template configurable

• distribution_type: auto_assign
